# AI Hub

Универсальный десктопный клиент для работы с несколькими AI-чатами в одном окне.

## Возможности

- Многоканальность — вкладки для DeepSeek, Qwen и GLM в одном приложении
- Темы оформления — Tokyo Night (по умолчанию), Dark, Light и другие
- Настройка акцентного цвета — выберите любой цвет подсветки
- Профили — создавайте несколько профилей с разными наборами вкладок
- Горячие клавиши — быстрые действия через клавиатурные комбинации
- Автозапуск — опциональный запуск при старте системы
- Кроссплатформенность — Windows и Linux

## Установка

### Windows

Скачайте установщик `AI Hub Setup X.X.X.exe` из раздела Releases и запустите его.

### Linux

Скачайте `ai-hub_X.X.X_amd64.deb` и установите:

```bash
sudo dpkg -i ai-hub_X.X.X_amd64.deb
```

## Сборка из исходников

```bash
git clone https://github.com/yourusername/ai-hub.git
cd ai-hub
npm install
npm run build:win
# или для Linux:
npm run build:linux
```

## Использование

1. Запустите приложение
2. Откройте настройки (значок шестерёнки)
3. Выберите тему и акцентный цвет
4. Добавьте нужные вкладки через профили
5. Авторизуйтесь в каждом сервисе

## Решение проблем

Проблема: Белая тема на странице входа
Решение: После авторизации программа сама подхватит тёмную тему.

Проблема: Вкладка не загружается
Решение: Проверьте подключение к интернету и обновите страницу.
